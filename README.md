# 简介

Wschat是一个简易的聊天系统，用户需要登录注册。并且已经绑定到了个人邮箱。

•一个聊天系统：

•用户可以注册登陆，设置个人信息，修改密码等。

•能添加好友，与好友交流。

•创建群聊，在群里畅聊。

•给他人空间留言

•能修页面配置，个性化
